618-MySQL Overview
618-MySQL is a Java applications for MySQL databases management; Java 1.8.0 or above needs to be installed on your device. 618-MySQL can be downloaded directly on to your desktop and used like any other desktop application. 618-MySQL can be used from a usb stick on any device equipped with MySQL databases. 618-MySQL has three separate display areas;
  1. A large display area for displaying the selected table and keeping track of actions performed.
  2. An area for displaying available databases, tables or columns; relating to the selected action.
  3. The coding area, where all the code for the selected action can be worked on.
618-MySQL has three separate modes of operation; Select Mode, Create Mode and Update Mode. Each mode has its own filing system; save your current code to file or open a file template and press execute.

618-MySQL Select Mode
With 618-MySQL Select Mode you can view and select MySQL databases and tables. After table selection, you are able to press the Enter key and display your table within the large 618-MySQL display area or you can use one or all of the four handy function keys; Where, And, Or and Order. That will add the following yellow inputs to the already displayed code.

SELECT
ID,
col1,
col2,
col3
FROM table1
WHERE column ><= value
OR column ><= value
AND column ><= value
ORDER BY column DESC / ASC
LIMIT 999
;

The gray code will be correct for the chosen table; you will only have to alter the newly added yellow code to suit your needs. The Where, Or, and And inputs can be used multiple times, maintaining correct code.

618-MySQL Create Mode
With 618-MySQL Create Mode you can create and drop databases, tables and triggers; again with handy function keys to simplify the process. An example of creating a new table follows.

CREATE TABLE
newTable
( ID INT(5) NOT NULL AUTO_INCREMENT,
 columnName INT(5) NOT NULL ,
columnName INT(5) NOT NULL ,
columnName INT(5) NOT NULL ,
 PRIMARY KEY (ID)
);

The gray code is presented after entering a table name and is ready to create newTable with the ID column only. The pink code was created by pressing the handy Column function key three times. There is a list of column Main Types and Constraints displayed in the adjoining text area to help when editing the the code.

618-MySQL Update Mode
With 618-MySQL Update Mode you have Update, Insert, Alter, Delete and CSV function keys. The Update Mode has by far the best time saving functions, that really need to be seen working; so please view my 618-tube video now.
