618-MySQL Overview
618-MySQL is a Java applications for MySQL databases management. 618-MySQL can be downloaded directly on to your desktop and used like any other desktop application. 618-MySQL can be used from a usb stick on any device equipped with MySQL databases. 618-MySQL has three main display areas;
  1. A large display area for displaying tables and keeping track of actions performed.
  2. An area for displaying databases, tables or columns; relating to the selected action.
  3. The coding area, where all current code for the selected action can be worked on.
618-MySQL has three separate modes of operation; Select Mode, Create Mode and Update Mode.

618-MySQL Select Mode
With 618-MySQL Select Mode you can view and select MySQL databases and tables. After table selection, you are able to press the Enter key and display your table within the large 618-MySQL display area or you can use one or all of the four handy function keys; Where, And, Or and Order. That will add the following Pink inputs to the already displayed code.

SELECT
ID,
col1,
col2,
col3
FROM table1
WHERE column ><= value
OR column ><= value
AND column ><= value
ORDER BY column DESC / ASC
LIMIT 999
;

The black code will be correct for the chosen table; you will only have to alter the newly added pink code to suit your needs. The Where, Or, and And inputs can be used multiple times, maintaining correct code.
