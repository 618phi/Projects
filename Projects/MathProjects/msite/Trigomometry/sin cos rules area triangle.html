Cosine and Sine Rules / Area of a Triangle

Question 1

Use the sine rule to find side p. Then use length p with the sine rule to find the length of side q. Use 4 decimal places when working out, but show your answers correct to 2 decimal places.



( p / sin(70) ) = ( 12 / sin(30) )
∴ p = ( 12 * sin(70) ) / sin(30)
∴ p = 22.5526
∴ side p = 22.55m Q = 180 - 70 - 30 = 80
∴ ( q / sin(80) ) = ( 22.5526 / sin(70) )
∴ q = 22.5526 * sin(80) / sin(70)
∴ q = 23.6354 ∴ side q = 23.64m
